Optimizing Muesli GmbH's Lead Time

Introduction

This was my inferential statistics project for the data analytics bootcamp I did with Spiced Academy. It consists of an Exploratory Data Analysis of the end-to-end delivery process (Lead Time) of a ficticious Muesli delivery company, and aplying some inferential statiscs as well as machine learning methods to try and predict the delivery times in future, based on historical data.

Roles

  • Data cleaning & Wrangling
  • Data Analytics
  • Data Visualisation

Tools

  • Python / Pandas
  • scikit-learn | Machine Learning
  • Seaborn / Matplotlib | Data Visualisation

Project Duration

  • Juli 2024
  • 2 days

Project Details

The goal of this project was to understand and extract the information from customer briefing, and based on it, to clean, prepare and combine multiple data sources, to than perform exploratory data analysis on. Expanded on the descriptive data analysis, I performed inferential statistics methods: an A/B test to check the Lead Time for each processing type and if there is some significant difference, and created a simple Linear regression model on just one factor which is the shipping mode to predict the Lead Times.

Gathering Samples and Bootstrapping

The two shipmodes were bootstrapped, meaning we increased the number of samples to make our experiment statistically significant.
Based of the size of the population, I decided that I needed at least 1000 datapoints to work with.

A/B Testing

Hypothesis 0: There is no difference between standard and express processing
Hypothesis 1: There is a difference between standard and express processing

Hypothesis 1 has been confirmed: high t-value (556), pvalue (= 0) means difference is statistically relevant, high degrees of freedom (1440) means the sample size is large enough.

Linear Regression

We predicted future Lead Times based on historical data with Linear Regression.

  • Step 1: convert our variable to numerical data. New column replacing standard proceeding with 0; and express with 1.
  • Step 2: defining the independent and dependent variables (X and y respectively):
    • X = shipment mode
    • Y = lead timea
  • Step 3: run the linear regression model and disclose the summary results.

The predicted Linear Regression model has a very low R^2 which means that:

  • Shipping mode only describes around 50% of the Lead Times.
  • We need to incorporate extra variables that more accurately explain the lead times.
  • This will yield a more precise Lead Time prediction.
